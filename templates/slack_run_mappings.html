How to read the labels

Each option shows the business label first and the stored field key in parentheses. Example: Wet THCA (g) (wet_thca_g). That keeps the screen readable while preserving stable field keys in the saved rules.

Current recommendation

  • Use Runs for mappings you need to preview today.
  • Use the destination field dropdown for other app areas whenever possible.
  • Use Custom field... only when a destination field is not listed yet.
