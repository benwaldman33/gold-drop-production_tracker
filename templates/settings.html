Maintenance

Recalculate Run Costs

Recomputes Cost/Gram for all historical runs using current biomass pricing and Cost Entries allocation. Use this after entering new costs or updating cost logic.

Backfill Historical Photo Records

One-time migration for historical approved field submissions. Creates Supplier Attachments for supplier/license photos and indexes all related images into the Photo Library for searchable audit history.

Pull Remote Site Caches

Fetches the configured manifest and summary payloads from every active remote site and refreshes the local cache used by aggregation endpoints.

Sync Slack channel history

Fetches messages via Slack conversations.history for each channel listed under Settings → Slack Integration → Channel history sync (up to 6). Only new messages (never seen Slack timestamps) are saved. Each channel stores a last-sync cursor after a successful run. Days back applies only when that channel has no cursor yet (first sync), or after you change/clear a channel row under Slack Integration (which resets its cursor). Otherwise a smaller “days back” does not shrink an already-ingested window. Invite the bot to each channel and grant OAuth scopes: channels:history, channels:read; for private channels add groups:history, groups:read.
